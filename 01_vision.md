# Antenna

**Status:** Pre-Alpha

## Vision

Antenna is an open-source Android application that provides a simple, efficient, responsive, practical, and intuitive interface for browsing community-driven discussion platforms.

Version 1 is designed for compatibility with Reddit while maintaining a clean, maintainable architecture and respecting the technical and policy constraints of the services it connects to.

Antenna exists to improve the user experience through thoughtful software design rather than unnecessary complexity. The project prioritises usability, performance, privacy, and transparency, allowing users to focus on the content and communities that matter to them.

The long-term objective is to build software that is reliable, maintainable, and enjoyable to use while remaining accessible to both users and contributors.

## Mission

To develop a modern, open-source Android application that demonstrates how community browsing software can be designed with clarity, efficiency, and respect for the user.

## Objectives

* Build a fast and responsive Android application.
* Provide a clean and intuitive browsing experience.
* Support anonymous browsing wherever technically possible.
* Minimise unnecessary complexity.
* Maintain a transparent and privacy-conscious design.
* Produce software that is well documented and easy to maintain.
* Establish a solid engineering foundation that can evolve over time.

## Design Philosophy

Antenna is designed around the belief that software should serve the user rather than compete for their attention.

Features should exist because they improve usability, not because they increase engagement.

Where multiple solutions exist, preference should be given to the simplest solution that satisfies the project's objectives.

## Version 1

Version 1 focuses on compatibility with Reddit and establishing a stable, maintainable foundation.

The goal of Version 1 is not to reproduce every available feature. Instead, it is to deliver a polished browsing experience that demonstrates the project's core philosophy and provides a platform for future development.

## Success Criteria

Antenna will be considered successful when it provides users with:

* A clean and responsive interface.
* A browsing experience that feels natural and intuitive.
* Reliable performance.
* Transparent behaviour.
* Confidence that the application is engineered with quality and maintainability as primary objectives.
