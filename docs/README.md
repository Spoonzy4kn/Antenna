# Antenna

**Status:** Pre-Alpha

Antenna is an open-source Android application designed to provide a simple, efficient, responsive, practical, intuitive, and transparent interface for browsing community-driven discussion platforms.

Version 1 targets compatibility with Reddit while maintaining a clean architecture, strong privacy principles, and a focus on usability.

The project is currently in the planning and documentation phase.
