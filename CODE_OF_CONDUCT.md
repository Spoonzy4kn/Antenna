# Code of Conduct

Antenna contributors are expected to keep discussion practical, respectful, and focused on improving the project.

## Expected behaviour

- Be respectful.
- Stay on topic.
- Give clear technical feedback.
- Assume good intent where reasonable.
- Prefer documented reasoning over personal preference.
- Respect privacy and security concerns.

## Unacceptable behaviour

- Harassment or abuse.
- Personal attacks.
- Publishing private information.
- Introducing intentionally harmful code.
- Ignoring security or privacy concerns.

## Enforcement

Maintainers may close issues, reject pull requests, remove comments, or block participants who disrupt the project.
